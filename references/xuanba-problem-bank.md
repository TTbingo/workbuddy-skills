### 格式说明

每道题包含：

- **好题自检**：5 条标准逐条 ✅/⚠️
- **辅导暗礁**：孩子常见的卡点 + 应对话术

### 1.1 0 为什么不能做除数？

**题干**：6÷0 = ? 为什么不能算？

**知识点**：1-2年级 · 数与运算 · 除法的意义

**方法论来源**：昍爸《苏格拉底式对话法》典型对话案例1

**苏格拉底引导链**：

```
Step 1: "6÷2=3，是不是相当于 6 减 2、减 2、减 2，减了 3 次等于零？"
        → 让孩子理解：除法就是反复减去除数

Step 2: "那 6÷0，就是 6 减 0、减 0……能减到 0 吗？要减多少次？"
        → 孩子发现：永远减不完，没有答案

Step 3: "那 0÷0 呢？0 减 0 等于 0，可以减 1 次，也可以减 2 次……答案是 1 还是 2？"
        → 孩子发现：答案可以是任何数，乱套了

让孩子自己说出结论：
"要么没结果，要么结果乱套，数学不允许这样，所以 0 不能做除数。"
```

**举一反三**：
- ⭐ L1：8÷0 = ? 12÷0 = ?（换数字）
- ⭐⭐ L2："把 10 块糖分给 0 个人，每人几块？"（换说法）
- ⭐⭐⭐ L3："如果规定一个数 a÷0 = 无穷大，那么无穷大 × 0 应该等于 a，但任何数 ×0=0……"（引发矛盾，揭示不可定义的本质）

**好题自检**：
| 标准 | 判断 |
|------|:---:|
| 有思维含量 | ✅ 孩子必须想，不能靠记忆 |
| 有多种解法 | ⚠️ 单一追问链，但追问本身就是多角度 |
| 可以变式 | ✅ 换数字/换说法/引矛盾 |
| 考查本质 | ✅ 考的是"除法的本质是反复减" |
| 有启发性 | ✅ 自己推出"数学不允许" |

**辅导暗礁**：
- 卡点：孩子说"老师说了 0 不能做除数，记住就行" → 回应："记住规则很重要，但知道为什么有这个规则，下次就不会忘了"
- 卡点：孩子理解不了"反复减" → 用实物：6 块糖每次拿走 2 块，拿 3 次没了；但每次拿走 0 块，永远拿不完

### 1.2 奇数偶数相加规律

**题干**：偶+偶=？奇+奇=？奇+偶=？不计算，能判断吗？

**知识点**：1-2年级 · 数与运算 · 奇偶性

**方法论来源**：昍爸《苏格拉底式对话法》典型对话案例2

**苏格拉底引导链**：

```
准备：小木棒若干根

Step 1: 把 6 根木棒两个两个摆 → 能配对完 → 偶数
        把 5 根木棒两个两个摆 → 多出一根 → 奇数

Step 2: "6+4 是什么数？"→ 算出来是 10（偶数）

Step 3: "不算结果，能判断吗？"
        → 孩子思考：6 摆完没剩，4 摆完没剩，两堆合并……肯定也没剩！

Step 4: "5+7 呢？"
        → 孩子用木棒想：两个孤单的木棒正好配对！

孩子自己总结：
- 偶+偶=偶（两堆都没剩）
- 奇+奇=偶（两个孤单的配对）
- 奇+偶=奇（一个孤单的，找不到对象）
```

**举一反三**：
- ⭐ L1：判断 13+15、24+37 的奇偶（换数字）
- ⭐⭐ L2："两个奇数相乘是奇是偶？三个奇数相加呢？"（换问法）
- ⭐⭐⭐ L3："任何偶数都可以写成 2×n，奇数可以写成 2×n+1，用字母证明奇+奇=偶"（代数化，5-6年级）

**好题自检**：
| 标准 | 判断 |
|------|:---:|
| 有思维含量 | ✅ 不计算，靠"配对"思维 |
| 有多种解法 | ✅ 实物摆 + 算式推 |
| 可以变式 | ✅ 乘法和连加都可延伸 |
| 考查本质 | ✅ 考的是"奇偶的本质是能不能配对" |
| 有启发性 | ✅ "原来不用算就知道答案" |

**辅导暗礁**：
- 卡点：孩子习惯先算再判断，说"我算出来就行了" → 回应："算出来当然可以，但如果你能不算就知道，说明你真的懂了奇偶的秘密"
- 情绪：木棒游戏本身就是趣味活动，通常不抵触

### 1.3 乘法是"快捷加法"

**题干**：3×4 和 4+4+4 有什么不同？为什么要有乘法？

**知识点**：1-2年级 · 数与运算 · 乘法意义

**方法论来源**：昍爸计算能力培养五阶段（第三阶段：表内乘法）

**苏格拉底引导链**：

```
Step 1: "3×4 是什么意思？"
        → 孩子：3 个 4 相加

Step 2: "如果让你算 100×4，你会写 100 个 4 连加吗？"
        → 孩子笑：太麻烦了！

Step 3: "所以乘法是什么？"
        → 孩子自己说：乘法就是把很多个相同的数加起来的快捷方式

Step 4: "那 3×4 和 4×3 一样吗？你能用积木摆给我看吗？"
        → 孩子摆：3 行 4 列 = 4 行 3 列 → 发现乘法交换律
```

**举一反三**：
- ⭐ L1：5×6 = ? 用加法验证（换数字）
- ⭐⭐ L2："超市的鸡蛋盒是 6×5，换个方向看变成 5×6，鸡蛋数变了吗？"（换场景）
- ⭐⭐⭐ L3："你发现了吗？3×4=4×3 不是巧合。换成任何两个数都对吗？"（引导发现乘法交换律）

**好题自检**：
| 标准 | 判断 |
|------|:---:|
| 有思维含量 | ✅ |
| 有多种解法 | ✅ 加法验证 + 积木摆 |
| 可以变式 | ✅ |
| 考查本质 | ✅ 乘法本质是相同加数的快捷加法 |
| 有启发性 | ✅ 自然引出交换律 |

**辅导暗礁**：
- 卡点：孩子说"我背九九乘法表就行" → 回应："背出来很快，但理解为什么 3×4=4×3，遇到没背过的（如 13×15）才不会慌"

### 2.1 排列问题：4 本练习册的排法

**题干**：书架上放 4 本不同的练习册，有多少种不同的排列方式？

**答案**：24 种（4×3×2×1）

**知识点**：3-4年级 · 统计与概率 · 排列初步

**方法论来源**：昍爸《化抽象为具象方法》案例——排列问题（建立模型思想）

**苏格拉底引导链**：

```
Step 1: "先从 3 本书开始排，有几种？"
        → 孩子动手摆出 6 种

Step 2: "现在再加一本橙色的。橙色的放第一本，后面三本有几种排法？"
        → 还是 6 种

Step 3: "橙色的放第二本呢？"
        → 还是 6 种

Step 4: "那橙色的可以放几个位置？每个位置后面有几种？"
        → 孩子发现：4 × 6 = 24
```

🚨 **关键**：不教 4! = 24 公式！让孩子自己经历"把新问题转化为熟题"的建模过程。

**举一反三**：
- ⭐ L1：5 本练习册有多少种排法？（换数字）
- ⭐⭐ L2："3 个人拍照站一排，有多少种站法？"（换场景）
- ⭐⭐⭐ L3："从中选 2 本借给同学，选法有几种？（不排顺序）"（变条件——排列 → 组合）

**好题自检**：
| 标准 | 判断 |
|------|:---:|
| 有思维含量 | ✅ 从 3→4 的递推建模 |
| 有多种解法 | ✅ 枚举 + 乘法定理 |
| 可以变式 | ✅ 排列→组合 |
| 考查本质 | ✅ 建模思想 |
| 有启发性 | ✅ "原来不用把所有排法都列出来" |

**辅导暗礁**：
- 卡点：孩子想直接列 24 种（太多易漏）→ 引导："先想简单的——3 本你会吗？"
- 暗礁：千万不要直接给公式！给孩子充分的实物操作时间

### 2.2 归纳推理：正三角形排一行的周长

**题干**：有 100 个边长为 1 的正三角形排成一行（相邻三角形共用一条边），图形的周长是多少？

**答案**：102

**知识点**：3-4年级 · 图形与几何 · 周长 + 数与运算 · 找规律

**方法论来源**：昍爸《三大推理能力》归纳推理案例

**苏格拉底引导链**：

```
Step 1: 1 个三角形 → 周长是多少？（3）

Step 2: 2 个三角形排一起 → 周长是多少？（4）
        → 为什么不是 6？因为中间那条边不算周长了

Step 3: 3 个三角形 → 周长？（5）
        4 个三角形 → 周长？（6）

Step 4: "你发现规律了吗？每加一个三角形，周长加几？"
        → 孩子：加 1

Step 5: "那 100 个三角形呢？"
        → 孩子：3 + 99×1 = 102
        或直接：100 + 2 = 102
```

**举一反三**：
- ⭐ L1：50 个正三角形排一行，周长？（换数字）
- ⭐⭐ L2：换成正方形排一行呢？正五边形呢？（换形状——见 3.2）
- ⭐⭐⭐ L3："如果有 n 个边长为 1 的正 m 边形排一行，周长是多少？用 m 和 n 表示"（代数推广）

**好题自检**：
| 标准 | 判断 |
|------|:---:|
| 有思维含量 | ✅ 从具体枚举到归纳规律 |
| 有多种解法 | ✅ 直接找规律 + 公式推导 |
| 可以变式 | ✅ 换形状、换数量 |
| 考查本质 | ✅ 考的是"公共边不参与周长"的本质 |
| 有启发性 | ✅ "原来不用数 100 个三角形" |

**辅导暗礁**：
- 卡点：孩子不理解"中间边不算周长" → 画图，用手沿着外围描一遍
- 卡点：孩子只想算不想画 → 坚持"从 1 个、2 个、3 个开始，不跳步"

### 2.3 类比推理：「弃 9 法」跨进制

**题干**：十进制中，297÷9 能整除的判断方法是 2+9+7=18 能被 9 整除。那么七进制中，435₍₇₎ ÷ 6₍₇₎ 能整除吗？有什么类似的判断方法？

**知识点**：3-4年级 · 数与运算 · 整除 + 进制初步

**方法论来源**：昍爸《三大推理能力》类比推理案例

**苏格拉底引导链**：

```
Step 1: "十进制里，怎么快速判断一个数能不能被 9 整除？"
        → 孩子：把各位数字加起来，看能不能被 9 整除

Step 2: "为什么？297 = 2×100 + 9×10 + 7 = 2×(99+1) + 9×(9+1) + 7
        = 2×99 + 9×9 + (2+9+7)"
        → 前面两项都是 9 的倍数，所以关键看 2+9+7

Step 3: "七进制里呢？6 在七进制里相当于十进制的 9（因为 7-1=6）"

Step 4: "435₍₇₎ = 4×7² + 3×7¹ + 5×7⁰ = 4×49 + 3×7 + 5
        = 4×(48+1) + 3×(6+1) + 5 = ...关键看 4+3+5"
        
让孩子的发现：进制变了，但"各位数字之和"的方法依然适用！
```

**举一反三**：
- ⭐ L1：七进制 324₍₇₎ ÷ 6₍₇₎ 能整除吗？（换数字）
- ⭐⭐ L2：五进制里判断能否被 4₍₅₎ 整除的方法是什么？（换进制）
- ⭐⭐⭐ L3："在 N 进制里，判断能被 N-1 整除的方法都是'各位数字之和'，为什么？"（一般性证明）

**好题自检**：
| 标准 | 判断 |
|------|:---:|
| 有思维含量 | ✅ 跨进制类比，深度思考 |
| 有多种解法 | ✅ 直接算 + 规律推广 |
| 可以变式 | ✅ 换进制、换除数 |
| 考查本质 | ✅ 数位原理 + 整除本质 |
| 有启发性 | ✅ "原来规律在另一个世界也成立" |

**辅导暗礁**：
- ⚠️ 此题适合 4 年级以上或对数学有浓厚兴趣的孩子。一般 3 年级可跳过。
- 卡点：孩子不熟悉进制 → 先用十进制讲透，再类比

### 2.4 鸡兔同笼（假设法 + 多种解法）

**题干**：笼子里有鸡和兔子若干只，共有头 14 个，腿 38 条，问鸡和兔子各有多少只？

**答案**：鸡 9 只，兔 5 只

**知识点**：3-4年级 · 数与运算 · 假设法 / 应用题

**方法论来源**：昍爸《算术思维 vs 方程思维》+ 配合 olympiad-cases.md 13 种解法

**苏格拉底引导链（假设法为主）**：

```
Step 1: "如果笼子里 14 只全部是鸡，一共多少条腿？"
        → 14×2 = 28 条

Step 2: "实际有 38 条腿，少了多少条？"
        → 38 - 28 = 10 条

Step 3: "为什么少了？因为我们把兔子当成鸡算了。每只兔子比鸡多几条腿？"
        → 4 - 2 = 2 条

Step 4: "少了 10 条腿，需要把几只鸡'变回'兔子？"
        → 10 ÷ 2 = 5 只

Step 5: "所以兔子几只？鸡几只？"
        → 兔 5 只，鸡 9 只

🚨 一定要验算：5×4 + 9×2 = 20 + 18 = 38 ✓
```

**举一反三**：
- ⭐ L1：头 20 个，腿 54 条（换数字）
- ⭐⭐ L2："停车场上停着汽车和摩托车共 24 辆，轮子共 86 个"（换场景）
- ⭐⭐⭐ L3："有 5 元和 10 元纸币共 20 张，共 145 元，各几张？"(硬币/纸币问题——同模型不同包装)

**好题自检**：全部 ✅（经典中的经典）

**辅导暗礁**：
- 卡点：孩子问"为什么要假设全是鸡" → 回应："你也可以假设全是兔，试试看？"（鼓励探索，不是指定唯一方法）
- 情绪：如果孩子用方程(x+y=14, 2x+4y=38)解出来了 → 肯定解法，但追问："不用 x、y 能做吗？"（训练算术思维）

### 2.5 盈亏问题

**题干**：老师给小朋友分糖果，如果每人分 3 颗，还剩 2 颗；如果每人分 4 颗，差 3 颗。问有多少个小朋友？多少颗糖？

**答案**：5 个小朋友，17 颗糖

**知识点**：3-4年级 · 数与运算 · 盈亏问题

**方法论来源**：昍爸《算术思维 vs 方程思维》盈亏问题

**苏格拉底引导链**：

```
Step 1: "两种分法，糖果总数一样吗？"
        → 一样

Step 2: "第一种每人 3 颗剩 2 颗，第二种每人 4 颗差 3 颗。每人多分了几颗？"
        → 4 - 3 = 1 颗

Step 3: "每人多分 1 颗，总共多用了多少颗糖？"
        → 剩的 2 颗用完了，还差 3 颗 = 2 + 3 = 5 颗

Step 4: "5 颗糖，每人 1 颗，能给几个人？"
        → 5 个人

Step 5: "那糖果总数呢？"
        → 5×3 + 2 = 17（或 5×4 - 3 = 17）
```

**举一反三**：
- ⭐ L1：每人 5 颗剩 1 颗，每人 6 颗差 4 颗（换数字）
- ⭐⭐ L2："宿舍分房间，每间 4 人多 2 人，每间 5 人少 1 间"（换场景）
- ⭐⭐⭐ L3："每人 a 颗剩 b 颗，每人 c 颗差 d 颗，用 a/b/c/d 表示人数"（代数模型化）

**好题自检**：全部 ✅

**辅导暗礁**：
- 卡点：孩子弄不清"剩 2 颗"和"差 3 颗"是什么关系 → 画图：画一排小人，上面标"多了 2"，下面标"差了 3"
- 卡点：孩子习惯于列方程 → 先让他列方程做对，再追问"有没有不用方程的方法？"

### 2.6 年龄问题

**题干**：爸爸今年 35 岁，儿子今年 5 岁。几年后爸爸的年龄是儿子的 3 倍？

**答案**：10 年后（爸爸 45 岁，儿子 15 岁）

**知识点**：3-4年级 · 数与运算 · 年龄问题

**方法论来源**：昍爸《算术思维 vs 方程思维》年龄问题

**苏格拉底引导链**：

```
Step 1: "爸爸和儿子的年龄差是多少？"
        → 35 - 5 = 30 岁

Step 2: "过多少年，这个差会变吗？"
        → 不变！不管过几年，爸爸永远比儿子大 30 岁

Step 3: "当爸爸年龄是儿子的 3 倍时，是什么意思？"
        → 爸爸 = 儿子 × 3

Step 4: "爸爸比儿子多几倍？"
        → 3 - 1 = 2 倍

Step 5: "这 2 倍对应的就是永远不变的 30 岁，那 1 倍是多少？"
        → 30 ÷ 2 = 15 岁（儿子的年龄）

Step 6: "儿子今年 5 岁，到 15 岁要几年？"
        → 10 年
```

**举一反三**：
- ⭐ L1：爸爸 40 岁，儿子 10 岁，几年前爸爸是儿子的 6 倍？（换数字 + 换方向）
- ⭐⭐ L2："爷爷今年 63 岁，孙子 7 岁，爷爷年龄是孙子的 5 倍时是哪一年？"（换人物）
- ⭐⭐⭐ L3："两人年龄和是 50 岁，甲比乙大 10 岁，甲几岁？"(和差问题——本质相同：利用不变量)

**好题自检**：全部 ✅（"年龄差不变"是不变量的经典教学案例）

**辅导暗礁**：

### 2.7 逆向思维：倒推法解题

**题干**：一个数加 5，乘 2，减 4 等于 10，这个数是多少？

**答案**：2

**知识点**：3-4年级 · 数与运算 · 逆运算

**方法论来源**：昍爸《六大数学思维》逆向思维

**苏格拉底引导链**：

```
Step 1: "最后一步是'减 4 等于 10'。减 4 之前是多少？"
        → 10 + 4 = 14

Step 2: "'乘 2 等于 14'。乘 2 之前是多少？"
        → 14 ÷ 2 = 7

Step 3: "'加 5 等于 7'。加 5 之前是多少？"
        → 7 - 5 = 2

Step 4: "验证一下：2 + 5 = 7，7 × 2 = 14，14 - 4 = 10 ✓"
```

**举一反三**：
- ⭐ L1：一个数减 3，乘 4，加 6 等于 30（换数字 + 换运算）
- ⭐⭐ L2："我今天花的钱比昨天多 5 元，昨天比前天少花 3 元，三天共花 50 元，今天花了多少？"（多步骤叙事型）

**好题自检**：
| 标准 | 判断 |
|------|:---:|
| 有思维含量 | ✅ 倒推是非线性思维 |
| 有多种解法 | ✅ 倒推法 + 列方程 |
| 可以变式 | ✅ 增加步骤/换场景 |
| 考查本质 | ✅ 逆运算的理解 |
| 有启发性 | ✅ "原来从后面往前想也是一种思路" |

**辅导暗礁**：
- 卡点：孩子搞混"减 4 等于 10 之前是 10+4 还是 10-4" → 用"时间机器"比喻：让时间倒流，每步做相反的运算

### 3.1 演绎推理：三角形内角和证明

**题干**：证明任意三角形的三个内角之和等于 180°。

**知识点**：5-6年级 · 图形与几何 · 三角形内角和

**方法论来源**：昍爸《三大推理能力》演绎推理案例

**苏格拉底引导链**：

```
Step 1: "先把三角形三个角剪下来拼在一起，你发现了什么？"
        → 拼成了一个平角（180°）——这是实验，不是证明

Step 2: "现在用证明的方法——过 C 点做 BA 的平行线。平行线有什么性质？"
        → 同位角相等、内错角相等

Step 3: "因为 BA//CE，所以 ∠DCE = 哪个角？"
        → ∠B（同位角）

Step 4: "∠ECA = 哪个角？"
        → ∠A（内错角）

Step 5: "那 ∠A + ∠B + ∠C = ?"
        → ∠ECA + ∠DCE + ∠ACB = 180°（平角）

Step 6（深追）: "同位角相等这条是从哪里来的？"
        → 欧几里得第五公设
        → 这才是演绎推理的"根基"
```

**举一反三**：
- ⭐ L1：用同样的方法证明四边形内角和（换对象）
- ⭐⭐ L2："过 A 点做 BC 的平行线，能证明吗？"（换辅助线做法——方法不唯一）
- ⭐⭐⭐ L3："如果不让用平行公设，三角形内角和还一定是 180°吗？"（非欧几何启蒙：球面三角形内角和 > 180°）

**好题自检**：全部 ✅

**辅导暗礁**：
- 卡点：孩子满足于"剪角拼成平角就是证明了" → 回应："实验让你相信，但不能说服所有人——万一你剪歪了呢？证明是从'不容怀疑的基本事实'出发的推理"
- ⚠️ 小学阶段不要求完全严格的公理化证明，重点在"证明意识"的培养

### 3.2 归纳推广：正五边形排一行周长

**题干**：承接 2.2(正三角形 → 周长 = n+2)。用同样的方法，探究 n 个正五边形排一行的周长。

**知识点**：5-6年级 · 图形与几何 · 周长规律推广

**方法论来源**：昍爸《三大推理能力》归纳推理——推广案例

**苏格拉底引导链**：

```
Step 1: "回顾三角形的规律：n 个三角形 → 周长 n+2"
        → 3 条边 - 1 条公共边 = 2 条外露边（每个三角形贡献）

Step 2: "五边形有 5 条边，两个拼在一起——几条边不参与周长？"
        → 1 条公共边

Step 3: "那 1 个五边形周长？2 个呢？"
        → 1 个：5；2 个：5+5-2=8

Step 4: "3 个？"→ 5+5+5-2-2 = 5×3-2×2 = 11
        4 个？→ 5×4-2×3 = 14

Step 5: "规律是什么？"
        → n 个五边形周长 = 5n - 2(n-1) = 3n + 2

Step 6: "如果是 m 边形呢？"
        → m×n - 2×(n-1) = (m-2)n + 2
```

**举一反三**：
- ⭐ L1：20 个正六边形排一行周长？（换多边形类型）
- ⭐⭐ L2："如果拼法变了——不是排一行，而是围成一圈，周长怎么算？"(变排列方式)
- ⭐⭐⭐ L3："m 边形拼成 k 行 n 列的网格，周长公式是什么？"（从一维到二维）

**好题自检**：全部 ✅ "从特例推广到一般公式"是归纳推理的范本

**辅导暗礁**：
- 卡点：n 个多边形时公共边数 = n-1 的理解 → 画图数：第 1 个和第 2 个之间 1 条、第 2 和第 3 之间 1 条……共 n-1 条
- 进阶衔接：这项能力直接对接初中"用字母表示数"

### 3.3 极值思维：绳子围矩形面积最大

**题干**：用一根长 20 米的绳子围成一个矩形，怎样围面积最大？最大面积是多少？

**答案**：正方形（边长 5 米），面积 25 平方米

**知识点**：5-6年级 · 图形与几何 · 周长面积 + 极值思维

**方法论来源**：昍爸《穷举法与极值思维》

**苏格拉底引导链**：

```
Step 1: "周长固定是 20 米，长+宽是多少？"
        → 10 米

Step 2: "把所有可能的长宽组合列出来："
        宽=1 长=9 → 面积=9
        宽=2 长=8 → 面积=16
        宽=3 长=7 → 面积=21
        宽=4 长=6 → 面积=24
        宽=5 长=5 → 面积=25

Step 3: "你发现什么规律？"
        → 长和宽越接近，面积越大！正方形时最大

Step 4: "为什么？用图形解释"
        → 把 5×5 正方形变成 6×4，相当于切掉 1 列移到旁边，
          多出来的部分比少掉的部分小

Step 5: "如果绳子长 24 米呢？如果长 100 米呢？"
        → 永远是正方形最大
```

**举一反三**：
- ⭐ L1：周长 36 米，最大面积？（换数字 → 边长 9，面积 81）
- ⭐⭐ L2："靠墙围矩形（一边靠墙不需绳），绳长 20 米，怎样面积最大？"(变条件——靠墙时不是正方形)
- ⭐⭐⭐ L3："用 20 米绳围一个一面开口的矩形，最大面积？"（边界条件变化）→ 衔接初中二次函数

**好题自检**：全部 ✅ 极值思维的经典入口

**辅导暗礁**：
- 卡点：孩子列到一半不想列了 → 鼓励："列完你就能发现一个惊人的规律"
- 衔接线 ⚡：这是小学→初中二次函数求最值的核心锚点（约束5）

### 3.4 三阶幻方的中心为什么是 5？

**题干**：在 3×3 的方格中填入 1-9，使每行、每列、每条对角线的和都相等。中心的数字必须是几？为什么？

**答案**：中心必须是 5

**知识点**：5-6年级 · 数与运算 · 幻方 + 整体思维

**方法论来源**：昍爸《三阶幻方的中心为什么要填 5？》

**苏格拉底引导链**：

```
Step 1: "1-9 的总和是多少？"
        → 1+2+3+4+5+6+7+8+9 = 45

Step 2: "一共 3 行，每行和相等，每行和是多少？"
        → 45 ÷ 3 = 15

Step 3: "经过中心有多少条线？"
        → 4 条：横、竖、两条对角线

Step 4: "这 4 条线上所有数字之和是多少？"
        → 4 × 15 = 60

Step 5: "这个 60 包含了哪些数字？"
        → 9 个数字各一次，但中心的数字被算了 4 次（实际只需要 1 次）
        → 所以：45 + 3×中心 = 60
        → 中心 = 5

让孩子自己发现：不是"因为 5 在中间所以幻方成立"，
而是"幻方必须成立，所以中心必须是 5"！
```

**举一反三**：
- ⭐ L1：如果把 1-9 换成 2-10，幻和是多少？中心填几？（换数字 → 中心=6）
- ⭐⭐ L2：四阶幻方呢？（1-16，换规模）
- ⭐⭐⭐ L3："用整体思维推导，n 阶幻方的中心有什么规律？"（一般性推广）

**好题自检**：全部 ✅ 整体思维的经典范本

**辅导暗礁**：
- 卡点：孩子不理解"中心的数被算了 4 次" → 画图：把 4 条线涂 4 种颜色，中心格子被涂了 4 层，其他格子只被涂了 1-2 层
- 这个解法的精髓是完全不依赖"怎么填"，只靠整体推理就锁定了中心——让孩子感受"整体思维"的力量

### 3.5 组成最大乘积的三位数×两位数

**题干**：用数字 1、2、3、4、5 各用一次，组成一个三位数和一个两位数，使乘积最大。这两个数分别是多少？

**答案**：431 × 52 = 22412（或 521 × 43，需比较）

**知识点**：5-6年级 · 数与运算 · 最大乘积 + 有序思维

**方法论来源**：昍爸《解题八步法》第六步"方法可扩展吗"中的例题 + 第七步"一题多解"

**苏格拉底引导链**：

```
Step 1: "要乘积最大，三位数的百位应该放什么数字？"
        → 放最大的（5 或 4）

Step 2: "两位数的十位呢？"
        → 也尽量大

Step 3: "试试 521 × 43 = ? 再试试 431 × 52 = ? 哪个大？"
        → 521×43 = 22403；431×52 = 22412 → 431×52 更大！

Step 4: "为什么百位放 4 比放 5 还大？"
        → 因为 52×400 比 43×500 带来的损失被其他位补偿了

Step 5（解题八步法第6步）: "如果换成 1-9 组成四位数×三位数呢？方法可扩展吗？"
        → 规律依然存在，但需要更多枚举
```

**举一反三**：
- ⭐ L1：用 2、3、4、5、6 组成最大乘积（换数字）
- ⭐⭐ L2："用 1-5 组成一个三位数×一个两位数，使乘积最小"（换方向）
- ⭐⭐⭐ L3："用 1-9 组成四位数×三位数使乘积最大"（换规模 → 解题八步法第6步验证方法可扩展性）

**好题自检**：全部 ✅（一题多解 + 可扩展 + 反直觉结果）

**辅导暗礁**：
- 卡点：孩子凭直觉说"5 放百位肯定最大"，结果发现 431×52 > 521×43 → 制造认知冲突，引导反思"为什么直觉错了？"
- 关键价值：这道题的核心不是算出答案，而是体验"直觉可能骗人 + 方法是否可扩展"

### 3.6 几何类比：正方形无限分割

**题干**：正方形边长 1，每次把右下角分成四等份并涂黑左上角那一个……无限重复。涂黑部分的总面积是多少？

**知识点**：5-6年级 · 图形与几何 · 等比数列求和（几何直观）

**方法论来源**：昍爸《三大推理能力》类比推理——几何类比案例

**苏格拉底引导链**：

```
Step 1: 第一次分：涂黑 1/4
        第二次分：在剩下的 3/4 中取 1/4 → 涂黑 3/4 × 1/4 = 3/16
        第三次分：再取剩下的 1/4 → 涂黑 3/4 × 3/4 × 1/4 = 9/64

Step 2: "总面积 = 1/4 + 3/16 + 9/64 + ..."

Step 3: "如果无限做下去，会怎么样？"
        → 无限等比数列求和 = (1/4) / (1 - 3/4) = 1

Step 4: "正方形总面积是 1，涂黑面积也是 1——那空白呢？"
        → 0！无限次之后每个角落都被涂黑了

Step 5: "如果把正方形换成正三角形，同样的操作，涂黑面积是多少？"
        → 类比推理：规律相同！
```

**举一反三**：
- ⭐ L1：每次涂黑 1/3（换比例）
- ⭐⭐ L2："换成正三角形，同样的操作"（换形状——类比迁移）
- ⭐⭐⭐ L3："如果每次涂黑比例不是 1/4 而是 r，总面积公式？"（公式化推广）

**好题自检**：全部 ✅ 极限思想的几何直观启蒙

**辅导暗礁**：
- ⚠️ 此题以几何直观为主，不要求 5-6 年级会算无穷等比数列——重点是感受"无限次操作后会发生什么"
- 用画图软件或纸笔画出前 3-4 次分割，让孩子直观感受"涂黑面积越来越大，空白越来越小"

### 3.7 斐波那契数列（兔子繁殖）

**苏格拉底引导链**：

Step 1: "第 1 个月：1 对（刚出生，不会生）"

### 3.8 汉诺塔问题

**题干**：有 3 根柱子，n 个大小不同的圆盘从大到小叠在 A 柱上。每次只能移动一个盘，且大盘不能放小盘上。把所有盘移到 C 柱，最少需要多少步？

**答案**：2^n - 1 步

**知识点**：5-6年级 · 综合 · 递归思维 + 找规律

**方法论来源**：昍爸《六大数学思维》递归思维

**苏格拉底引导链**：

```
Step 1: 1 个盘 → 几步？（1 步）
        2 个盘 → 几步？(3 步：小→B, 大→C, 小→C)

Step 2: "3 个盘呢？"
        → 试出来：7 步

Step 3: "你发现规律了吗？1→1, 2→3, 3→7"
        → 每次 ×2+1：1×2+1=3, 3×2+1=7

Step 4: "为什么是 ×2+1？"
        → 要把最大的盘从 A 移到 C，需要先把上面 n-1 个盘移到 B(2^(n-1)-1 步)
        → 移动最大的盘（1 步）
        → 再把 n-1 个盘从 B 移到 C(2^(n-1)-1 步)
        → 总共：2×(2^(n-1)-1) + 1 = 2^n - 1
```

**举一反三**：
- ⭐ L1：4 个盘最少几步？（换数字：15 步）
- ⭐⭐ L2："如果改成 4 根柱子呢？最少步数怎么变？"（变条件）
- ⭐⭐⭐ L3："证明 64 个盘按每秒移一次需要 5800 亿年"（感受指数爆炸）

**好题自检**：全部 ✅ 递归思想的殿堂级案例

**辅导暗礁**：
- 卡点：孩子只想动手试，不想找规律 → 先让他试 1、2、3 个盘，记录结果，自然引出规律
- 情绪：3 个盘以上不用真动手（可用在线演示），重点放在找规律

### 3.9 圆的面积公式推导

**题干**：圆面积的公式 S=πr² 是怎么来的？为什么和 π、r² 有关？

**知识点**：5-6年级 · 图形与几何 · 圆面积

**方法论来源**：昍爸《12 个核心公式》第 10

**苏格拉底引导链**：

```
Step 1: "把一个圆切成很多个相等的扇形（像切蛋糕）"

Step 2: "把这些扇形交错拼起来，像什么形状？"
        → 像长方形！

Step 3: "这个'长方形'的长是多少？"
        → 半个圆的周长 = πr

Step 4: "宽是多少？"
        → 圆的半径 r

Step 5: "面积呢？"
        → πr × r = πr²

Step 6（深追）: "切的份数越多，越接近长方形——如果切无限多份呢？"
        → 就是真正的长方形（极限思想）
```

**举一反三**：
- ⭐ L1：半径 3cm 的圆面积？（换数字）
- ⭐⭐ L2："把圆切成同心圆环后展开，能推导面积公式吗？"(换方法)
- ⭐⭐⭐ L3："球体积公式 4/3πr³ 能类似推导吗？"（三维类比）

**好题自检**：全部 ✅

**辅导暗礁**：
- 这是极限思想的第一次正式接触——不必深讲极限，重点是"切得越细越像长方形"的直观感受
- 可以让孩子真的用纸剪一个圆来拼

### 3.10 三角形面积公式推导

**题干**：三角形面积 = 底×高÷2，为什么是"÷2"？

**知识点**：5-6年级 · 图形与几何 · 三角形面积

**方法论来源**：昍爸《12 个核心公式》第 9

**苏格拉底引导链**：

```
Step 1: "两个完全一样的三角形，能拼成什么形状？"
        → 平行四边形（或长方形）

Step 2: "平行四边形的面积怎么算？"
        → 底 × 高

Step 3: "那一个三角形的面积呢？"
        → 平行四边形的一半 = 底 × 高 ÷ 2

Step 4: "为什么一定是'两个完全一样的三角形'才能拼？"
        → 引导孩子理解"对应关系"

Step 5: "如果不拼平行四边形，还有其他推导方法吗？"
        → 割补法：从长方形割掉一个直角三角形
```

**举一反三**：
- ⭐ L1：底 6cm 高 4cm 的三角形面积（换数字）
- ⭐⭐ L2："用割补法（从长方形中割掉三角形）推导"（换方法）
- ⭐⭐⭐ L3："梯形面积公式又是怎么来的？和三角形面积推导有什么共同点？"（迁移：都是拼成平行四边形）

**好题自检**：全部 ✅

**辅导暗礁**：
- 卡点：孩子满足于记住公式 → 回应："记住公式当然好，但知道为什么÷2，换成梯形的(上底+下底)×高÷2你也能自己推出来"
- 画图！一定要画图！两个三角形拼成平行四边形是核心图像

### 4.1 分数基本性质

**题干**：为什么 1/2 = 2/4 = 3/6？"分子分母同乘同除同一个数，分数值不变"是为什么？

**知识点**：3-5年级 · 数与运算 · 分数

**方法论来源**：昍爸《12 个核心公式》第 11

**苏格拉底引导链**：

```
Step 1: "1/2 是什么意思？"
        → 把 1 个蛋糕切成 2 份，取 1 份

Step 2: "如果切成 4 份，取 2 份——和刚才取的多少一样吗？"
        → 一样！每份小了但取的份数多了

Step 3: "切成 6 份取 3 份呢？"
        → 还是一样

Step 4: "你发现了吗？只要切的份数翻 n 倍，取的份数也翻 n 倍，量就不变"
        → 这就是分子分母同乘同除的直观意义
```

**举一反三**：
- ⭐ L1：判断 3/5 = 9/15 是否正确（换数字）
- ⭐⭐ L2："为什么约分后分数值不变？"(换方向：从扩分到约分)
- ⭐⭐⭐ L3："如果分子分母加同一个数（如 1/2 → 2/3），值变了吗？"(反例——揭示"只有乘除才不变")

**好题自检**：全部 ✅

**辅导暗礁**：
- 卡点：孩子死记"同乘同除" → 回应："你知道为什么不能同加吗？蛋糕切成 2 份取 1 份，和切成 3 份取 2 份，一样大吗？"

### 4.2 乘法分配律（长方形面积模型）

**题干**：(a+b)×c = a×c + b×c。为什么乘法要"分配"？用图形解释。

**知识点**：3-5年级 · 数与运算 · 运算律

**方法论来源**：昍爸《12 个核心公式》第 5（最重要的公式）

**苏格拉底引导链**：

```
Step 1: 画一个长方形，长 = a+b，宽 = c
        面积 = (a+b) × c

Step 2: 从中间画一条竖线，把长方形分成两个：
        左边：长 a × 宽 c = a×c
        右边：长 b × 宽 c = b×c

Step 3: "总面积等于左边加右边吗？"
        → 当然！
        → (a+b)×c = a×c + b×c

Step 4: "为什么要分配？"
        → 因为可以把大问题拆成小问题分别算，再合起来
```

**举一反三**：
- ⭐ L1：(3+5)×4 = 3×4 + 5×4，用数字验证
- ⭐⭐ L2："(a+b)×(c+d) = ? 用长方形面积模型推导"（推广到两项×两项）
- ⭐⭐⭐ L3："12×13 = 12×(10+3) = 120+36 = 156——这是口算的核心技巧，就是这个模型"（应用意识）

**好题自检**：全部 ✅（昍爸称为"12 个公式中最重要的一个"）

**辅导暗礁**：
- 卡点：孩子只会背公式不会用图形理解 → 必须画图！长方形面积模型是最强大的理解工具
- 这是小学→初中代数（多项式乘法）的核心锚点

### 4.3 几何辅助线思考（通用方法，非单一题目）

**核心问题**：辅助线不是"凭空想到"的，而是从"条件到结论缺了什么"推导出来的。

**知识点**：3-6年级 · 图形与几何 · 辅助线策略

**方法论来源**：昍爸《搞定平面几何：辅助线是怎么想出来的？》

**苏格拉底式四步追问**（任意几何题通用）：

```
Step 1: 这道题要我求什么？（目标明确）
Step 2: 现在有哪些条件？（把已知全摆出来）
Step 3: 条件和目标之间，缺了什么？（找辅助线方向）
Step 4: 缺的那个关系，有没有办法搭一座桥？（构造辅助线）

典型追问话术：
- "你能把这道题的已知条件一个个说出来吗？"
- "从已知到答案，缺了哪一步？"
- "这道题和做过的哪道题像？"
- "如果把这道题的数字改小一点，你还会做吗？"
```

**六种辅助线类型速查**：

| 类型 | 什么时候用 | 辅导追问 |
|------|-----------|---------|
| 延长线 | 条件不够用 | "把点往右延一点，能看到什么？" |
| 平行线 | 需传递角/线段关系 | "哪两条线应该平行？平行会怎样？" |
| 中点线 | 涉及中点 | "中点把线段分成两段，什么关系？" |
| 垂线 | 需要高或直角 | "这里需要一个直角，怎么做？" |
| 连对角线 | 四边形问题 | "对角连线能帮上什么忙？" |
| 构造等腰 | 涉及等腰/等边 | "有没有相等的边？能构造等腰吗？" |

**辅导暗礁**：
- 不期望孩子一开始就会自己想辅助线——先用追问引导，慢慢过渡到孩子自己问自己这四步
