# LaTeX 数学公式输出规范

> 在辅导中输出公式时，遵循以下规范，确保公式清晰、可读、年级适配。

## 使用时机

| 场景 | 是否用 LaTeX | 说明 |
|------|:-----------:|------|
| 学生模式 · 1-2年级 | ❌ 不用 | 用汉字或口语描述："8加7等于15" |
| 学生模式 · 3-4年级 | ⚠️ 少量用 | 仅用于展示算式：`$45 + 38 = 83$` |
| 学生模式 · 5-6年级 | ✅ 可以用 | 可展示分数 `$\frac{3}{4}$`、方程 `$x + 5 = 12$` |
| 家长模式 · 所有年级 | ✅ 可以用 | 家长需要看到精确的数学表达，全面使用 |

## 格式规则

| 规则 | 正确 ✅ | 错误 ❌ |
|------|---------|---------|
| **简单算式用行内公式** | `$8 + 7 = 15$` | `$$8 + 7 = 15$$`（不必独占一行） |
| **多步推导用块级公式** | `$$14 \times 2 = 28$$` | 把多步推导挤在一行 |
| **分数用 `\frac{}{}`** | `$\frac{3}{4}$` | `3/4`（低年级可用，高年级用 frac） |
| **乘号用 `\times`** | `$3 \times 4 = 12$` | `3*4` 或 `3x4` |
| **除号用 `\div`** | `$12 \div 3 = 4$` | `12/3`（仅在讲分数概念时用斜线） |

## 反例：什么时候不要用 LaTeX

| 场景 | 错误做法 | 正确做法 |
|------|---------|---------|
| 给一年级孩子讲"凑十法" | `$8 + 7 = 8 + 2 + 5 = 10 + 5 = 15$` | "8加7，先想8加几等于10？8加2等于10。7拿走2还剩5，10加5等于15。" |
| 孩子刚学分数概念 | `$\frac{2}{3} > \frac{1}{2}$` | "把一块饼分成3份拿2份，比分成2份拿1份，哪个多？" |
| 孩子情绪已经崩了 | 继续用 LaTeX 输出公式 | 丢弃公式，用实物类比和画图沟通 |
