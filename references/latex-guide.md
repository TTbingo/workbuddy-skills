# 数学符号书写与排版规范

> **这份规范的唯一目标：让输出的数学式子，像试卷一样规范、像课本一样易读。**
>
> LaTeX 只是实现手段。读者不需要知道 `\frac` 是什么——他们只需要看到竖着写的分数，分子在上、分母在下，清楚明白。

---

## 一、给谁看，就怎么排

不同读者对数学符号的接受度不同。排版的第一原则不是"准确"，而是"对方能不能一眼看懂"。

### 1.1 受众分层

| 读者 | 排版策略 | 例子 |
|------|---------|------|
| 🧒 **1-2年级学生** | 不用符号，用口语描述 | "8加7，先想8加几等于10……" |
| 🧒 **3-4年级学生** | 只排算式，不排复杂符号 | `45 + 38 = 83` |
| 🧒 **5-6年级学生** | 可排分数、方程 | ⅔ > ½，x + 5 = 12 |
| 🧑 **初中生** | 全量符号，含根号、指数 | √2，x²，½（分数竖排） |
| 🧑 **高中生** | 全量符号，含极限、积分、向量 | lim f(x)（x→0 写在下方），∫ₐᵇ f(x)dx，a⃗ |
| 👨‍👩‍👦 **家长** | 全量符号，精确表达，可截图转发 | 始终用最清晰的排法 |

### 1.2 核心判断标准

排完一个式子之后，问自己：

> **一个不懂 LaTeX 的普通家长，看到这个式子，能不能像看课本一样直接读出来？**

能 → 通过。不能 → 重新排。

## 二、排版习惯

### 2.1 行内 vs 独行

| 情况 | 排版方式 | 说明 |
|------|---------|------|
| 单个简单算式 | 嵌在文字里 | "因为 `8 + 7 = 15`，所以进位了" |
| 多步推导 | 独占一行 | 让读者看清每一步怎么变 |
| 重要公式（定理、结论） | 独占一行、居中 | 视觉上突出，像课本里的公式框 |
| 两个以上算式并列对比 | 各占一行，对齐 | 方便眼睛上下扫 |

### 2.2 多步推导：竖式对齐

每步一行，等号对齐，像课本例题：

```
原式 = 3 × (2 + 4)
     = 3 × 6
     = 18
```

**不要**把多步挤成一行：~~`3 × (2 + 4) = 3 × 6 = 18`~~（读者要自己拆步骤，累）

### 2.3 方程组：左大括号，右对齐

```
⎧ 2x + y = 7
⎨
⎩ x - y = 2
```

### 2.4 数与单位之间空一格

✅ `30 分钟` `5 厘米` `100 元`
❌ `30分钟` `5厘米` `100元`

## 三、符号约定

以下约定覆盖日常辅导中最常用的符号。每一条的出发点都是**读者看着不费劲**，而不是"LaTeX 里怎么写"。

### 3.1 四则运算

| 运算 | 写法 | 不用 |
|------|------|------|
| 加法 | `3 + 4 = 7` | — |
| 减法 | `9 − 5 = 4` | — |
| 乘法 | `3 × 4 = 12` | `3*4`、`3x4`（容易和字母 x 混淆） |
| 除法 | `12 ÷ 3 = 4` | `12/3`（只在讲分数时用斜线） |
| 点乘（高中向量） | `a · b` | — |

### 3.2 分数：竖着写，不斜着写

✅ 竖式分数：分子在上，分数线横穿，分母在下
❌ 斜线分数：`3/4`（小学高年级以上一律不用斜线，除非是和文字嵌在一起的随口提法）

家长看到竖着的分数，一眼就知道"上面是被分的，下面是份数"——这是十几年课本训练出来的直觉。

### 3.3 指数与下标

✅ 指数写在右上角：`x²`，`2³ = 8`
✅ 下标写在右下角：`a₁`，`Sₙ`

注意：`x^2` 是代码，`x²` 才是给人看的。

### 3.4 根号：完整包裹被开方数

✅ √(2) 含义清晰，根号把整个被开方数框住
❌ `√2+1` 到底是 √2 + 1 还是 √(2+1)？读者会犹豫

### 3.5 比较符号

✅ `3 > 2`（大于号），`5 < 8`（小于号）
✅ `x ≥ 0`，`y ≤ 10`
✅ `a ≠ b`

### 3.6 括号层级

从内到外：小括号 → 中括号 → 大括号

✅ `3 × [(2 + 4) ÷ 2]`
❌ `3 × ((2 + 4) ÷ 2)`（内外都用小括号，层级看不清）

### 3.7 常见函数名：正体，小写

✅ `sin x`，`log₂ 8`，`lim f(x)`
❌ `sinx`（挤在一起），`SIN x`（大写不对）

### 3.8 特殊符号

| 符号 | 写法 | 说明 |
|------|------|------|
| 度 | `90°` | 角度、温度都用 |
| 圆周率 | `π` | 直接写希腊字母 π |
| 无穷 | `∞` | — |
| 约等于 | `≈` | 不是 `~` 或波浪线 |
| 因为/所以 | `∵` / `∴` | 初中几何常用 |

## 四、反例：什么时候不要堆符号

符号是工具，不是目的。以下场景堆符号不如说人话：

| 场景 | ❌ 堆符号 | ✅ 说人话 |
|------|---------|---------|
| 一年级讲"凑十法" | `8 + 7 = 8 + 2 + 5 = 10 + 5 = 15` | "8加7，先想8加几等于10？8加2等于10。7拿走2还剩5，10加5等于15。" |
| 孩子刚接触分数概念 | `⅔ > ½` | "把一块饼分成3份拿2份，比分2份拿1份，哪个多？" |
| 孩子情绪已经崩了 | 继续排公式 | 丢开符号，用实物、画图、类比来沟通 |
| 对话中随口一提 | 排版一个豪华公式 | 直接说"三分之二"就行，不用排版 |

## 附录：LaTeX 速查表

> 以下是将上面规范翻译成 LaTeX 代码的速查。**这不是规范本身——规范在正文里。这里只是实现参考。**

### A.1 基础符号

| 你想要的输出 | LaTeX 代码 |
|-------------|-----------|
| 行内算式 `8 + 7 = 15` | `$8 + 7 = 15$` |
| 独行公式（居中） | `$$3 \times 4 = 12$$` |
| 分数 ³⁄₄ | `$\frac{3}{4}$` |
| 乘号 × | `\times` |
| 除号 ÷ | `\div` |
| 点乘 · | `\cdot` |
| 指数 x² | `$x^2$` |
| 下标 a₁ | `$a_1$` |
| 平方根 √x | `$\sqrt{x}$` |
| n 次根 ⁿ√x | `$\sqrt[n]{x}$` |

### A.2 比较与括号

| 你想要的输出 | LaTeX 代码 |
|-------------|-----------|
| ≥ | `\ge` 或 `\geq` |
| ≤ | `\le` 或 `\leq` |
| ≠ | `\neq` |
| ≈ | `\approx` |
| ± | `\pm` |

### A.3 高中常用

| 你想要的输出 | LaTeX 代码 |
|-------------|-----------|
| 求和 Σ | `$\sum_{i=1}^{n}$` |
| 极限 lim | `$\lim_{x \to 0}$` |
| 积分 ∫ | `$\int_a^b f(x)\,dx$` |
| 无穷 ∞ | `\infty` |
| 向量 a⃗ | `$\vec{a}$` |
| 角度 90° | `$90^\circ$` |
| 希腊字母 α β γ θ π | `\alpha \beta \gamma \theta \pi` |

### A.4 放弃 LaTeX

在微信等不支持 LaTeX 渲染的环境里，用以下替代：

| 符号 | 替代写法 |
|------|---------|
| ² | 直接输入 Unicode：`x²` |
| ³ | 直接输入 Unicode：`2³` |
| ½ | 直接输入 Unicode：`½` |
| × | 中文输入法打"乘号" |
| ÷ | 中文输入法打"除号" |
| √ | 中文输入法打"根号" |
| π | 中文输入法打"pai" |

**微信输出优先走 Unicode 替代方案**，确保接收方看到的是渲染好的数学符号，而不是一堆 `\frac` 源码。
