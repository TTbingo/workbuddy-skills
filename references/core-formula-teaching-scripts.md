## 1. 加法交换律 a+b=b+a

| 步骤 | 话术 |
|------|------|
| **场景引入** | "你有3块糖，妈妈又给了5块，和先有5块再加3块，总数一样吗？" |
| **算理讲解** | 加法是合并——不管先加哪一堆，合起来的总数不变。用小木棒演示：3根+5根 和 5根+3根，摆出来都是8根。 |
| **提问链** | T1→"3+5和5+3都等于几？" T2→"为什么结果一样？你能用木棒摆出来吗？" T3→"如果换成 27+38 和 38+27，你能不算直接判断结果一样吗？" |
| **错误预判** | 孩子会认为"必须先算前面的"——这是因为竖式训练形成的惯性。用"合并两堆糖"破除此惯性。 |
| **延伸检验** | L1 换数字：15+28=? 28+15=?；L2 换说法："先吃3个再吃5个"和"先吃5个再吃3个"；**检验**："能讲给我听为什么一样吗？" |

## 2. 加法结合律 (a+b)+c=a+(b+c)

| 步骤 | 话术 |
|------|------|
| **场景引入** | "去超市买了3样东西：面包8元、牛奶12元、苹果5元。先算面包+牛奶再加苹果，和先算牛奶+苹果再加面包，总价一样吗？" |
| **算理讲解** | 加法是逐步合并——只要每一步都正确，合并顺序不影响总数。关键价值：**凑整计算**——(8+12)=20先算最方便。 |
| **提问链** | T1→"8+12+5 先算哪两个最方便？" T2→"8+12=20，20+5=25 ——如果先算12+5=17，17+8=25，哪个快？" T3→"为什么要学结合律？它在生活中有用吗？"（答：凑整心算更快） |
| **错误预判** | 孩子可能把"结合律"和"交换律"搞混。区分口诀："交换律=换位置，结合律=加括号"。 |
| **延伸检验** | L1：27+36+14 → 先算 36+14=50；L2：买4样东西的账单凑整；**检验**："能出一道要用结合律凑整的题吗？" |

## 3. 乘法交换律 a×b=b×a

| 步骤 | 话术 |
|------|------|
| **场景引入** | "教室里每排坐6个人，有4排——6×4=24人。如果每排坐4个人，有6排——4×6=24人。总人数一样吗？" |
| **算理讲解** | 画点子图（6行×4列 = 4行×6列 = 24个点），旋转90°后，行变列、列变行，但总点数不变。这就是乘法交换律的几何本质。 |
| **提问链** | T1→"6×4和4×6都等于24，为什么？" T2→"用点子图解释：6行×4列转90°变成什么？" T3→"如果让你出一道题考同学乘法交换律，你会怎么出？" |
| **错误预判** | 孩子可能从加法交换律"照搬"过来以为"乘法当然可以交换"——但除法不能！要强化"乘法交换有几何道理"，不是拍脑袋。 |
| **延伸检验** | L1：7×9=? 9×7=?；L2："3个5"和"5个3"画点子图对比；**检验**："能用图形解释为什么3×8=8×3吗？" |

## 4. 乘法结合律 (a×b)×c=a×(b×c)

| 步骤 | 话术 |
|------|------|
| **场景引入** | "一箱饮料有4排，每排6瓶，装了3层——一共多少瓶？先算每层多少再乘层数，和先算每列多少再乘列数，结果一样吗？" |
| **算理讲解** | 用积木搭长方体的体积模型：4×6×3——(4×6)×3=24×3=72 和 4×(6×3)=4×18=72 结果相同。关键价值：**凑整简化**——25×4=100 是最经典的凑整。 |
| **提问链** | T1→"25×17×4 先算哪两个最省事？" T2→"25×4=100，100×17=1700 ——如果先算 17×4=68，再算 25×68 呢？哪个快？" T3→"结合律和交换律有什么区别？能用一个例子说明吗？" |
| **错误预判** | 混淆交换律和结合律——"换位置"vs"加括号"。用对比练习区分。 |
| **延伸检验** | L1：8×23×125 → 先算 8×125=1000；L2：长方体装箱问题改编；L3："如果把乘法结合律推广到小数还成立吗？试试 0.25×4×1.7"；**检验**："能出一道需要用结合律凑整的题吗？" |

## 5. ⚠️ 乘法分配律 (a+b)×c=a×c+b×c（最重要！）

| 步骤 | 话术 |
|------|------|
| **场景引入** | "一个长方形花园，左边种花区域长5米，右边种菜区域长3米，宽都是4米。总面怎么算？" |
| **算理讲解** | **面积模型是最佳工具**：画一个宽4米的大长方形，拆成左边(5×4)和右边(3×4)两个小长方形。(5+3)×4=32 = 5×4+3×4=20+12=32。分配律的本质是"把乘法分给加法里的每一个"。 |
| **提问链** | T1→"大长方形宽4，长(5+3)=8，面积=8×4=32。拆成两个长方形呢？" T2→"为什么(5+3)×4=5×4+3×4？用面积图解释。" T3→"分配律反过来用叫什么？（提取公因数）12×7+8×7=(12+8)×7=140 ——生活中哪里见过？" |
| **错误预判** | ⚠️ **最高频错误**：把分配律当成"把乘法分配进括号里每个数"，但只乘了a没乘b——如 2×(3+5)=2×3+5。纠正：用面积图，让孩子看见"两个小长方形都必须乘同样的宽"。 |
| **延伸检验** | L1：6×(4+9)；L2："两个年级共买故事书：三年级6个班每班买12本，四年级4个班每班买12本——共买多少？两种算法对比"；L3："如果分配律的c在左边：(a+b)×c 和 c×(a+b)一样吗？为什么？"；**检验**："能用两种不同方法证明分配律吗？" |

## 6. 路程=速度×时间

| 步骤 | 话术 |
|------|------|
| **场景引入** | "从家骑车去学校，每分钟骑250米，骑了12分钟到——家和学校距离多远？" |
| **算理讲解** | 速度是"每分钟走多远"，时间是多长时间，乘起来就是总路程。**不是加法**——不是1分钟+2分钟……加12次250，而是"250的12倍"，用乘法更高效。实际上 250×12 = 250×(10+2) = 2500+500 = 3000。 |
| **提问链** | T1→"每分钟250米，12分钟一共？为什么用乘法不用加法？" T2→"如果速度是每分钟x米，走了t分钟，路程怎么表示？" T3→"路程÷时间=速度，路程÷速度=时间——这三个关系你能写出来吗？为什么它们互逆？" |
| **错误预判** | 单位混淆：把分钟和小时直接乘。纠正："分钟要统一！要么全用分钟，要么全用小时。" |
| **延伸检验** | L1：换数字；L2：追及问题变式（两人同向不同速）；**检验**："如果只知道路程和时间，怎么求速度？能出一道这样的题吗？" |

## 7. 正方形面积=a²

| 步骤 | 话术 |
|------|------|
| **场景引入** | "一个小正方形地砖边长30厘米——它的面积是多大？如果用1平方厘米的小方格来铺，需要多少个？" |
| **算理讲解** | 用小方块摆满正方形——每行a个，有a行，一共a×a=a²个。a²读作"a的平方"，本质是二维的数量。**区别于周长**：周长是4a（一维的边），面积是a²（二维的面）。 |
| **提问链** | T1→"边长1cm的正方形面积是多少？边长2cm呢？" T2→"为什么边长2cm的正方形需要4个1cm²的小方块？（画图）" T3→"如果正方形边长扩大2倍，面积扩大几倍？为什么不是2倍？（面积扩大4倍，因为宽和高都2倍了）" |
| **错误预判** | ⚠️ 面积和周长混淆：把 a² 和 4a 搞混。纠正：用图区分——"周长是围一圈的线，面积是铺满的面"。 |
| **延伸检验** | L1：边长5cm→面积；L2："一个正方形周长是24cm，面积是多少？"（先求边长再面积）；L3："用12个1cm²的小正方形能拼出几种不同的长方形？最大边长是多少？"；**检验**："能讲给我听为什么面积用平方单位吗？" |

## 8. 长方形面积=a×b

| 步骤 | 话术 |
|------|------|
| **场景引入** | "教室的地面长8米、宽6米——需要多少块边长1米的正方形地砖才能铺满？" |
| **算理讲解** | 用小方块摆满长方形——每行b个（对应长），有a行（对应宽），一共a×b个。面积=a×b 的本质就是"每行的个数×行数"。 |
| **提问链** | T1→"长5cm、宽3cm的长方形，画格子需要几行？每行几个？" T2→"如果长方形长扩大2倍、宽不变，面积怎么变？" T3→"正方形面积=a² 和长方形面积=a×b 是什么关系？（正方形是长=宽的长方形特例）" |
| **错误预判** | 长和宽搞反——认为"谁大谁就是长"。纠正："长和宽只是名字，a×b和b×a结果一样（乘法交换律），所以不纠结哪个叫长。" |
| **延伸检验** | L1：换数字；L2："用24个1cm²的小方块能拼出多少种长方形？（列举所有可能的长宽组合）"；L3："如果只告诉你长方形周长是20cm，面积最大是多少？（正方形时最大）"；**检验**："能出一道求长方形面积的题吗？要包含单位换算。" |

## 9. 三角形面积=ah÷2

| 步骤 | 话术 |
|------|------|
| **场景引入** | "拿一张长方形的纸，沿对角线剪开——得到什么？两个一模一样的三角形！" |
| **算理讲解** | ⚠️ **必须用图形演示！** 用纸剪或用画图：两个完全相同的三角形可以拼成一个平行四边形。平行四边形的面积=底×高，而一个三角形是它的一半，所以÷2。**强调**：÷2 不是因为"三角形比长方形小"，而是因为两个三角形才能拼成一个完整的长方形/平行四边形。 |
| **提问链** | T1→"一个三角形和一个等底等高的平行四边形，面积有什么关系？" T2→"用两个三角形拼平行四边形试试——你能证明三角形面积=底×高÷2吗？" T3→"如果三角形不是直角三角形，高怎么找？所有三角形都能这样证明吗？" |
| **错误预判** | ⚠️ **最经典错误**：忘记÷2！直接底×高。纠正：每次讲都先让孩子"拼一拼"两个三角形，强化"÷2来自两个拼一个"。 |
| **延伸检验** | L1：换数字；L2："一个三角形底10cm，高未知，面积是25cm²——高是多少？"（逆推）；L3："用三种不同方法推导三角形面积公式（拼图法/割补法/坐标系）"；**检验**："能在纸上剪两个三角形拼给我看吗？" |

## 10. 圆的面积=πr²

| 步骤 | 话术 |
|------|------|
| **场景引入** | "披萨店的12寸披萨和两个6寸披萨，哪个更多？直觉说两个6寸=12寸？算算看！" |
| **算理讲解** | ⚠️ **用极限思想**：把圆切成很多个相等的扇形，像切披萨。切得越细，拼起来越接近一个长方形。这个"长方形"的长≈圆周长的一半(πr)，宽≈半径(r)，面积≈πr×r=πr²。这不是严格证明，但让孩子感受"从曲到直"的极限过渡思想。 |
| **提问链** | T1→"圆的面积公式里有r²——为什么是r²而不是r？" T2→"如果把圆切成8块拼，像什么？切16块呢？切100块呢？" T3→"披萨问题：12寸直径的面积是 π×6²=36π，两个6寸=2×π×3²=18π ——差了整整一倍！为什么直觉错了？" |
| **错误预判** | 把面积公式和周长公式搞混：πr² vs 2πr。纠正："周长是绕一圈（一次幂），面积是铺满整个面（二次幂）"。 |
| **延伸检验** | L1：已知半径求面积；L2："半径扩大2倍，面积扩大几倍？"(4倍，因为r²)；L3："如果一个圆的面积是50.24cm²(π≈3.14)，半径是多少？"；**检验**："为什么圆的面积公式里有π？π从哪里来？" |

## 11. 分数基本性质

| 步骤 | 话术 |
|------|------|
| **场景引入** | "一块蛋糕平均切成4份，取1份是1/4。同样的蛋糕切成8份，取2份是2/8——哪个多？" |
| **算理讲解** | 分数基本性质：分子分母同时乘（或除以）同一个不为0的数，分数大小不变。本质：1/4 和 2/8 表示的是同样大小的量，只是"切的份数"和"取的份数"同步变化。用纸条对折对折再对折来演示。 |
| **提问链** | T1→"1/2 和 2/4 一样大吗？怎么证明？"（画图或折纸）T2→"3/5 的分子分母都×2，变成6/10——为什么还等于3/5？" T3→"分数基本性质有什么用？（约分/通分/比较大小）如果分子分母同时加同一个数，分数大小会变吗？试试 1/2→(1+1)/(2+1)=2/3≠1/2" |
| **错误预判** | ⚠️ 孩子会类推"加减也保持大小"：1/2=(1+1)/(2+1)=2/3≠1/2！纠正：必须用图对比——"加不是按比例缩放，乘才是"。 |
| **延伸检验** | L1：写出 3/4 的 3 个等值分数；L2："比较 5/8 和 7/12 的大小——先通分再比"；L3："如果分子分母同时加同一个正数，分数值是变大还是变小？证明你的结论。"；**检验**："能出一道关于分数基本性质的判断题吗？" |

## 12. 比例基本性质 ad=bc

| 步骤 | 话术 |
|------|------|
| **场景引入** | "冲果汁：浓缩汁和水按1:4的比例——用了3杯浓缩汁需要多少水？3:12=1:4——对不对？怎么验证？" |
| **算理讲解** | 两个比相等即 a:b=c:d → 交叉相乘相等：a×d=b×c。意义：外项积=内项积。用天平模型解释：天平左边是"a:b"这个比，右边是"c:d"这个比，两边平衡→交叉相乘也平衡。 |
| **提问链** | T1→"2:3=4:6 —— 2×6=3×4 都是12，为什么？" T2→"如果 3:5=x:15，x等于几？你是怎么算的？" T3→"比例基本性质和分数基本性质是什么关系？（比例是两个相等的比，比可以写成分数形式）" |
| **错误预判** | 混淆"比"和"比值"——认为 2:3 是一个数。纠正："2:3不是0.666……，它表示的是'2份对3份'的关系。" |
| **延伸检验** | L1：解比例 4:x=12:15；L2："地图比例尺1:500000，图上3cm代表实际多少公里？"；L3："比例尺问题：如果把地图放大到原来的2倍，新比例尺是多少？"；**检验**："能用天平模型解释为什么 ad=bc 吗？" |

## 附录：公式教学黄金追问 5 句

每讲完一个公式，必做这套追问（对应约束16四层检验法）：

| # | 追问 | 检验层次 |
|:--:|------|:---:|
| 1 | "这个公式是研究什么的？" | ① 会做 |
| 2 | "你能用图形把这个公式解释给我听吗？" | ② 会讲 |
| 3 | "这个公式在生活中哪里见过？" | ②→③ |
| 4 | "如果没有这个公式，有没有别的办法？" | ③ 会变 |
| 5 | "你能出一道要用这个公式的题吗？" | ④ 会出 |
