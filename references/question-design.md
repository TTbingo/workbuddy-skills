# 出题规范

当用户要求"出题"时，按以下规范执行。

## ⚠️ 出题前必做（两步）

**第一步：查例题库**
- 出题前，**必须先读** `references/exam-bank.md`，查看是否有同年级、同题型的题目可参考

**第二步：标注考查点（强制）**
- 每道题出完后，**必须**在题目标题下方标注考查点：
  ```
  【考查点】核心素养：XXX｜知识点：XXX｜难度：⭐～⭐⭐⭐
  ```
- 核心素养从课标2022版中选择：数感/量感/符号意识/运算能力/推理意识/空间观念/几何直观/数据意识/模型意识/应用意识/创新意识

## ⚠️ 两条铁律（必须遵守）

1. **绝对不能写"如图"**：除非同时提供了实际图片，否则严禁使用"如图所示""看图""如图"等任何暗示图形的表述。几何题如果需要图形，必须用**文字描述**让孩子自己画出来。
2. **默认不提供答案**：除非用户明确要求，否则**不要**在出题后附上参考答案。只需要提供"思考提示"，让孩子先做。

## 出题模板

```
【题目】
（具体题目，难度匹配年级）

【考查点】核心素养：XXX｜知识点：XXX｜难度：⭐～⭐⭐⭐
（🔴 必填，不可省略）

【思考提示】（可折叠）
 提示1：先想一想……
 提示2：如果不会，试试……

【做完验算】（可选）
 验算方向提示，不给结果
```
