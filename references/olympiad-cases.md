## 一、鸡兔同笼问题

### 问题原型

> 笼子里有鸡和兔子若干只，共有头14个，腿38条，问鸡和兔子各有多少只？
> **答案**：鸡9只，兔子5只

---

### 13种解法详解

#### 解法1：列表法（适合3年级初步接触）

| 鸡 | 兔 | 腿数 | 判断 |
|----|----|------|------|
| 0 | 14 | 56 | 太多 |
| 3 | 11 | 50 | 太多 |
| 5 | 9 | 46 | 太多 |
| 7 | 7 | 42 | 太多 |
| **9** | **5** | **38** | **✓** |

> 💡 **辅导要点**：引导孩子跳跃式列表，不要逐个枚举（效率低），快速定位。

---

#### 解法2：画图法（适合形象思维强的孩子）

1. 假设14只全部是鸡 → 画14只鸡（每只2条腿）
2. 数一数：共28条腿
3. 比实际少：38-28=10条腿
4. 给其中5只"补腿"（每只补2条）→ 变成5只兔子
5. **结论：兔子5只，鸡9只**

> 💡 **辅导要点**：用图形代替抽象数字，对低年级尤其有效。

---

#### 解法3：金鸡独立法（有趣且深刻）

1. 让所有鸡用一只脚站立，兔子用两只后脚站立
2. 地上的脚数变为原来的一半：38÷2=19（只脚）
3. 此时：鸡的脚数=头的数量，兔的脚数=2×头的数量
4. 用总脚数（19）减去总头数（14）= **5** → 这就是兔子的数量
5. **结论：兔子5只，鸡9只**

> 💡 **辅导要点**：这个方法让孩子从"算"的角度跳出来，用"观察"直接得出结果。讲解后可以问："为什么除以2？"（深化位值原理的理解）

---

#### 解法4：吹哨法

1. 吹第一声哨：鸡兔各抬一只脚，剩下38-14=24只脚
2. 吹第二声哨：各再抬一只脚
3. 此时：鸡坐在地上（0只脚），兔子还有2只脚站着
4. 站着的10只脚全是兔子的：10÷2=5只兔子
5. **结论：兔子5只，鸡9只**

---

#### 解法5：假设法（假设全鸡）

1. 假设全是鸡：14×2=28条腿
2. 比实际少：38-28=10条腿
3. 每把一只鸡换成一只兔子，增加2条腿
4. 需要换：10÷2=5只（换成兔子）
5. **结论：兔子5只，鸡9只**

---

#### 解法6：假设法（假设全兔）

1. 假设全是兔子：14×4=56条腿
2. 比实际多：56-38=18条腿
3. 每把一只兔子换成一只鸡，减少2条腿
4. 需要换：18÷2=9只（换成鸡）
5. **结论：鸡9只，兔子5只**

---

#### 解法7-11：趣味特异功能法（适合激发兴趣）

| 解法 | 形象描述 | 核心思路 |
|------|---------|---------|
| 7 | 鸡翅变腿法 | 假设翅膀变腿 → 多出18条=鸡的翅膀数 → 18÷2=9只鸡 |
| 8 | 鸡飞兔立法 | 鸡飞起来，兔子用后腿站 → 38-28=10只全是兔子后腿 |
| 9 | 劈开半兔法 | 兔子劈开变"半兔"（1头2脚）→ 总数19只，多出5只=兔子数 |
| 10 | 砍足法（古法） | 砍脚后：19只脚，独角鸡1头1脚，双脚兔1头2脚 → 19-14=5只兔 |
| 11 | 耍兔法 | 兔子耍酷抬前脚 → 兔子10÷2=5只 |

> 💡 **辅导要点**：趣味解法是为了让孩子**对数学产生兴趣**，可以在孩子学有余力时使用。但如果只停留在"趣味"层面，而不追问"为什么这样对"，就只是记住了技巧，没有提升思维。

---

#### 解法12-13：方程法（适合5-6年级）

**设鸡为x只**：
```
2x + 4(14-x) = 38
2x + 56 - 4x = 38
2x = 18
x = 9（鸡）
兔 = 14 - 9 = 5（只）
```

**设兔为x只**：
```
4x + 2(14-x) = 38
4x + 28 - 2x = 38
2x = 10
x = 5（兔）
```

> 💡 **辅导要点**：方程法是**最高层次的抽象**。先用画图法/假设法让孩子有直观理解，再用方程法做形式化提升。不建议跳过直观阶段直接教方程。

### 变式练习题

1. **鸡兔共49只，腿100条**，问鸡兔各几只？（答案：鸡48只，兔1只）
2. **鸡兔共20本（头），总脚56只**，鸡兔各几只？
3. **鸡蛋邮票问题**：用10元买50张邮票，4角和8角各几张？（答案：25张4角，25张8角）
4. **玻璃瓶问题**：运1000个玻璃瓶，完好1个得1.5元，损坏1个倒扣2元，最后得1458元，损坏多少个？（答案：16个）

## 二、和差问题

### 核心公式

```
大数 = （和 + 差）÷ 2
小数 = （和 - 差）÷ 2
```

### 记忆口诀

> 和加上差，越加越大，除以2便是大的；
> 和减去差，越减越小，除以2便是小的。

### 经典例题

#### 例1：两班人数

> 甲乙两班共有学生98人，甲班比乙班多6人，求两班各有多少人？

**苏格拉底式辅导**：
- "98人和6人差，这两个数相加再除以2，得到的是哪个班？"→ 引导发现"大数=（和+差）÷2"
- "那小数怎么算？"→ 引导从98-6=92，或直接用公式

**解答**：
```
甲班 = （98 + 6）÷ 2 = 52（人）
乙班 = 98 - 52 = 46（人）
```

#### 例2：长方形的长宽

> 长方形长与宽之和为18厘米，长比宽多2厘米，求面积。

**辅导要点**：不直接套公式，先问"长和宽谁大？"→ 引导画出长方形标注

**解答**：
```
长 = （18 + 2）÷ 2 = 10（cm）
宽 = （18 - 2）÷ 2 = 8（cm）
面积 = 10 × 8 = 80（cm²）
```

## 三、和倍问题

### 核心公式

```
较小数 = 和 ÷ （倍数 + 1）
较大数 = 和 - 较小数
```

### 经典例题

#### 例1：杏树和桃树

> 果园里有杏树和桃树共248棵，桃树的棵数是杏树的3倍，求各多少棵？

**苏格拉底式辅导**：
- "桃树是杏树的3倍，如果杏树是1份，桃树是几份？"→ 引导画线段图
- "一共是几份？"→ 1+3=4份
- "248棵平均分成4份，每份是多少？"→ 248÷4=62（杏树）
- "那桃树呢？"→ 62×3=186（棵）

#### 例2：父子年龄（和生活紧密相关）

> 爸爸36岁，儿子9岁，爸爸年龄是儿子的几倍？

**辅导要点**：这道题反过来问——先问孩子"能算出倍数吗？"→ 36÷9=4倍。然后延伸："爸爸比儿子大几岁？明年呢？后年呢？"→ 引导发现**年龄差不变**这一重要规律。

## 四、差倍问题

### 核心公式

```
较小数 = 差 ÷ （倍数 - 1）
较大数 = 较小数 × 倍数
```

### 经典例题

#### 例1：果树问题

> 桃树是杏树的3倍，且桃树比杏树多124棵，求各多少棵？

**辅导要点**：关键在于找到"1倍量"。先问"如果杏树是1份，桃树是几份？"→ 3份。追问"桃树比杏树多几份？"→ 2份。

**解答**：
```
杏树 = 124 ÷ (3-1) = 62（棵）
桃树 = 62 × 3 = 186（棵）
```

#### 例2：父子年龄

> 爸爸比儿子大27岁，爸爸的年龄是儿子的4倍，求父子二人今年各是多少岁？

**辅导要点**：**年龄问题三不变**——年龄差始终不变，年龄倍数逐年变化。此题用差倍公式一步到位。

**解答**：
```
儿子 = 27 ÷ (4-1) = 9（岁）
爸爸 = 9 × 4 = 36（岁）
```

## 五、盈亏问题

### 核心公式

```
人数 = （盈 + 亏）÷ 两次分配差
```

### 盈亏问题的三种类型

| 类型 | 条件 | 公式 |
|------|------|------|
| 盈亏型 | 一次有余，一次不足 | 人数 = （盈+亏）÷ 分配差 |
| 盈盈型 | 两次都有余 | 人数 = （盈₁-盈₂）÷ 分配差 |
| 亏亏型 | 两次都不足 | 人数 = （亏₁-亏₂）÷ 分配差 |

### 经典例题

#### 例1（盈亏型）

> 幼儿园把一箱饼干分给小班和中班的小朋友，平均每人分6块；只分给中班，每人可多分4块。如果只分给小班，每人分多少块？

**辅导要点**：先让孩子理解"平均分给所有人"和"只分给一部分人"的差异——分的人越少，每人分到的越多。

**解答**：
```
设总人数 = x人，总饼干数不变
6x = （x-？) × (6+4) → 需要知道中班人数

此题实际解法：
总饼干 = 盈（饼干多出来的部分）/ 分配差 = ...
```

#### 例2（盈盈型）

> 把一些积木分给小朋友，每人分2个则多20个；每人分3个则差40个。幼儿园有多少小朋友？一共多少个积木？

**辅导要点**：每人多拿1个，需要的积木多20+40=60个 → 小朋友有60人

**解答**：
```
人数 = (20 + 40) ÷ (3-2) = 60（人）
积木数 = 2×60 + 20 = 140（个）
```

## 六、植树问题

### 三种情形

| 情形 | 条件 | 公式 |
|------|------|------|
| 两端都植树 | 两头都种 | 棵树 = 段数 + 1 |
| 一端植树 | 一头种一头不种 | 棵树 = 段数 |
| 两端都不植 | 两头都不种 | 棵树 = 段数 - 1 |
| 封闭图形 | 围成圈 | 棵树 = 段数 |

### 经典例题

#### 例1（两端都植）

> 在一条长80米的路两边种树，每隔10米种一棵，一共要种多少棵？

**辅导要点**：先问"80米，每10米一段，一共几段？"→ 8段。再问"两端都种，棵树和段数什么关系？"→ 棵树=段数+1

**解答**：
```
段数 = 80 ÷ 10 = 8（段）
棵树（一边）= 8 + 1 = 9（棵）
两边共种 = 9 × 2 = 18（棵）
```

#### 例2（两端都不植）

> 在100米的大桥两侧安装路灯，每隔20米装一盏，两端不装，一共装多少盏？

**解答**：
```
段数 = 100 ÷ 20 = 5（段）
一边 = 5 - 1 = 4（盏）
两边 = 4 × 2 = 8（盏）
```

## 七、行程问题

### 核心公式

```
路程 = 速度 × 时间
速度 = 路程 ÷ 时间
时间 = 路程 ÷ 速度
```

### 经典例题

#### 例1：相遇问题

> 甲、乙两人从相距240千米的两地同时出发相向而行，甲速度50千米/小时，乙速度30千米/小时，几小时后相遇？

**辅导要点**：两人同时走1小时，一共走50+30=80千米。问孩子"走多少个80千米就到240千米？"

**解答**：
```
1小时共走：50 + 30 = 80（千米）
相遇时间 = 240 ÷ 80 = 3（小时）
```

#### 例2：往返相遇（复杂）

> 甲、乙两车同时从A、B两地出发，第一次在离A地80千米处相遇。相遇后继续行驶到达对方出发点后返回，第二次在离B地60千米处相遇。求A、B两地距离。

**辅导要点**：这是行程问题中的难点，需要用到**比例关系**。引导孩子发现：第一次相遇时，甲走了80千米，乙走了"全程-80千米"。从第一次相遇到第二次相遇，两车共走了2倍全程。

**解答**：
```
设全程为 S 千米
第一次相遇：甲走80千米，乙走 S-80 千米
从第一次相遇到第二次相遇：甲乙共走 2S 千米

速度比 = 80 : (S-80)
第二次相遇时，乙走的路程 = 80 + 60 = 140（千米）

用比例：
S-80 : 80 = (80+60) : S-80 → S = 150 千米
```

## 八、三大推理能力与奥数题型的对应

| 推理类型 | 适合题型 | 核心思维 |
|---------|---------|---------|
| **类比推理** | 和差倍问题、盈亏问题 | 找到题目间的共同结构，套用已知方法 |
| **归纳推理** | 植树问题、数列找规律 | 从几个特殊例子归纳出一般规律 |
| **演绎推理** | 行程问题、方程应用 | 从已知条件严密推导未知 |

### 在辅导中的具体应用

**类比推理**：
- 讲完"和差问题"后，让孩子类比做"和倍问题"
- 问："这两个题有什么相同的地方？""不同在哪里？"

**归纳推理**：
- 讲植树问题时，先让孩子枚举：小段数→棵树，归纳出三段公式
- 问："如果我把段数从5变成10，棵树怎么变？"

**演绎推理**：
- 行程问题：用画线段图，逐一分析每个阶段的已知和未知
- 训练孩子"有根有据地写每一步"，而非靠感觉猜答案
