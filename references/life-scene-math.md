# 🎯 生活场景数学化（8 个对照场景）

> 核心理念：四年级前孩子抽象思维未成熟，**生活场景让数学"看得见、摸得着"**。
> 操作原则：先找生活中的对应物，再做数学抽象。

## 场景对照表

| 生活场景 | 数学概念 | 启发方式 | 适用年级 |
|---------|---------|---------|:------:|
| 分蛋糕 | 分数 | "平均分成几份，取其中几份" | 2-3 |
| 超市比价 | 小数、除法、单价 | "哪个更便宜？怎么算单价？" | 3-4 |
| 装书架 | 等差数列 | "每层多几本，一共几层？" | 3-5 |
| 乒乓球忘了谁发球 | 周期问题 | "轮流发球，第N球谁发？" | 4-5 |
| 锅里的包子圈数 | 圆周长 | "外圈比内圈多几个？" | 4-6 |
| 坐位体前屈量负数 | 负数概念 | "基准线以下就是负数" | 5-6 |
| 数地砖 | 几何、乘法 | "行×列=总数" | 2-3 |
| 零花钱记账 | 四则运算 | "收入-支出=余额" | 1-4 |

## 操作流程

```
Step 1: 识别数学概念 → 查表找对应场景
Step 2: 用场景"先问生活问题"→ 再抽象到数学
Step 3: 如果已有场景不完全匹配 → 换一个相近的 → 或创造新场景
```

## 化抽象为具象三步

```
具象（实物摆放：分蛋糕、摆地砖）
  ↓
半具象（画图：线段图、格子图）
  ↓
抽象(符号：分数 a/b、乘法 ×)
```

## if-then 规则

| 条件 | 动作 |
|------|------|
| 辅导对象 ≤ 4 年级 | 从 8 个场景中选 ≥1 个嵌入引导脚本 |
| 学生卡在抽象理解 | 优先查本表寻找可感知的对应物 |
| 家长问"怎么教XX概念" | 先查本表 → 选场景 → 给场景化引导步骤 |
| 场景不完全匹配 | 基于"先生活→后抽象"原则自创场景 |

## 反例

- ❌ 直接讲 a/b 的符号意义 → 孩子只记住了符号
- ✅ 先问"蛋糕分3份取1份怎么表示"→ 再说"我们把它写成 1/3"
