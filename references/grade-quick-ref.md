# 年级对应知识点速查

> 📚 **完整知识库**：`references/comprehensive-knowledge-base.md`
> 📝 **全学段索引**：`references/exam-bank.md`（瘦身心引文件，~120行），指向各学段参考资料
> 📝 **经典案例库**：`references/case-studies.md`（12个经典启发案例）

| 年级 | 核心知识点 | 关键算理 |
|------|-----------|---------|
| 一年级 | 20以内加减法、认识10 | 凑十法（理解数的分拆，而不是死记） |
| 二年级 | 表内乘除法、100以内加减 | 乘法本质是"相同加数的加法"，除法是"平均分" |
| 三年级 | 多位数乘除法、分数初步 | 竖式的位值原理，分数是"整体的几份" |
| 四年级 | 大数运算、小数加减 | 小数点对齐 = 位值对齐 |
| 五年级 | 分数四则、方程入门 | 分数乘除的图形理解，方程是"等量关系" |
| 六年级 | 比例、百分数、统计 | 比例是对应关系，百分数是特殊分数 |
| 初一（七年级） | 有理数、整式加减、一元一次方程 | 负数扩展数轴，方程是"求未知量的相等关系" |
| 初二（八年级） | 三角形全等、轴对称、整式乘除、因式分解 | 全等是"变换不变性"，因式分解是"乘法公式的逆运算" |
| 初三（九年级） | 一元二次方程、二次函数、圆 | 二次函数是"变化率的可视化"，圆是"到定点距离相等的点的集合" |
| 高一（必修） | 集合、函数、平面向量、数列 | 函数是"输入→输出"的映射，向量是"量+方向" |
| 高二（选必） | 立体几何、解析几何、导数、概率统计 | 导数是"瞬时变化率"，解析几何是"用代数方法研究几何" |
| 高三（选修+复习） | 复数、数学建模、专题深化 | 复数是"实数的二维扩展"，数学建模是"用数学语言描述现实" |
