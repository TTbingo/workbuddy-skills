# 🧮 中小学数学AI辅导老师（小初高一体化）

> **不只做题，教你怎么教。** 胡小群 + 昍爸 + 子贤老师 + 塞利格曼的中小学数学辅导方法论（小初高一体化+积极心理学+认知重构），装进 AI Agent 里。

孩子问"3×4 为什么等于 12？"——你不再只能说"背下来"，这个 Skill 会用苏格拉底式追问、小木棍、分蛋糕和面积模型，让 ta 自己"想到"答案。

## 这是什么？

一个 AI Agent Skill，把中国中小学数学教育领域的实战派名师——**胡小群**（原复旦附中实验班名师）、**昍爸**（中科院计算机博士）、**子贤老师**（「教练式父母训练营」积极心理学+学习策略）和**马丁·塞利格曼**（积极心理学之父，《教出乐观的孩子》《真实的幸福》）——的方法论，编码成了可执行的辅导引擎。覆盖**小学+初中+高中**全学段，支持跨阶段知识点螺旋上升衔接辅导。

它不只是"解题工具"，而是一个**教学哲学系统**：

| 不是 | 而是 |
|------|------|
| ❌ 直接给答案 | ✅ 苏格拉底式追问，引导孩子自己想出来 |
| ❌ 只讲"怎么算" | ✅ 深挖"为什么这么算"（算理 > 算法） |
| ❌ 机械刷题 | ✅ 吃透一道经典题，胜过刷 100 道 |
| ❌ 死记公式 | ✅ 从具象到抽象，理解公式的推导过程 |

---

## 核心差异化

- **双模式设计**：自动识别学生/家长身份，切换完全不同的辅导策略（学生用苏格拉底追问，家长用辅导脚本）
- **方法论品牌化**：胡小群"算理优先+小初高一体化" + 昍爸"解题八步法+六大思维+三大推理" + 子贤老师"积极心理学+学习策略" + 塞利格曼"解释风格+ABCDE认知重构" + 课标2022 核心素养体系，五合一
- **19 个 reference 资产**：全学段例题库、全量知识库、奥数经典题库、小初高课标原文、方法论文档5份（胡小群/昍爸/昍爸进阶/子贤老师/塞利格曼）、失败模式库（G1-G24）、经典案例库、工具规范4份（LaTeX/出题/追问算法/P3机制）
- **完整 SOP**：身份识别 → 情况了解 → 类型判断 → 启发式辅导 → 算理讲解 → 练习布置 → 反馈总结
- **持续进化机制（P3）**：每次真实辅导后记录失败/成功案例，定期提炼新 gotcha 并更新 gotchas.md，让 Skill 从"理论最优"进化到"实战最优"

---

## P3 定期回顾机制

这个 Skill 不会停在"理论完美"。每次真实辅导后，它会：

1. **收集反馈**：主动问"这次辅导有什么特别的地方吗？"
2. **提炼失败模式**：分析是否有新的失败模式（不在 G1-G24 中的）
3. **更新 gotchas.md**：把新失败模式写成 gotcha，补充到失败模式库
4. **检查约束有效性**：回顾是否遵守了「品味编码」中的 if-then 约束

**触发词**："辅导完了"、"这次辅导怎么样？"、"有什么需要改进的吗？"

每次对话开始时，Skill 会先确认你是**学生本人**还是**家长/老师**，然后自动切换对应模式。

## 安全边界：这个 Skill 不会做什么

| ❌ 不会做 | 原因 |
|-----------|------|
| **直接给答案** | 核心原则：不灌答案，用启发 |
| **批评孩子** | 错了 = 发现了学习机会 |
| **超前压榨** | 四年级前不强行用抽象方式教公式 |
| **堆砌口诀** | 口诀学得越多，理解越少 |
| **写"如图"** | 没有图就绝不说"如图"，用文字引导孩子自己画 |
| **替代老师/家长** | 定位是辅导工具，不改作业、不替代教学关系 |
| **出题后直接给答案** | 默认只给思考提示，不直接提供答案 |

## 示例输出

### 🧒 学生模式示例

> **孩子**："鸡兔同笼，14个头，38条腿，我不会做。"

> **AI**："我们先不急着做题。你想象一下：如果笼子里全是鸡，那14只鸡有多少条腿？"

> **孩子**："28条。"

> **AI**："对！但题目说有38条腿，差了多少条？"

> **孩子**："差了10条。"

> **AI**："很好。那每把一只鸡换成一只兔子，腿会多几条？"

> **孩子**："……多2条！"

> **AI**："所以要补上10条腿，需要换几只？"

> **孩子**："5只！所以兔子5只，鸡9只！"

---

### 👨‍👩‍👦 家长模式示例

> **家长**："孩子三年级，算 4×6 总是算错，怎么教？"

> **AI**：给出完整辅导脚本——

> **题目解析**：4×6 的本质是"4个6相加"或"6个4相加"，关键算理是"乘法是相同加数加法的简便写法"。

> **常见错误分析**：三年级孩子可能还在机械记忆阶段，容易混淆 4×6 和 6×4 的结果。需要先确认孩子是否理解"乘法的意义"。

> **辅导话术（直接对孩子说）**：
> 1. "4×6 是什么意思？你能用加法写出来吗？"（让他写 4+4+4+4+4+4）
> 2. "这里面有几个4？"（6个）
> 3. "所以 4×6 和 6×4 的结果一样吗？"（引导发现乘法交换律）
> 4. "如果再算 7×4，你怎么算？"（检验迁移能力）

> **延伸建议**：出一道变式题——"5×7 和 7×5 的结果一样吗？为什么？"
